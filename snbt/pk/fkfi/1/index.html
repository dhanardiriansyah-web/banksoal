1.
Jawaban: A. $3x + 2$ 1. Dasar Teori

Komposisi fungsi $(f \circ g)(x)$ berarti memasukkan fungsi $g(x)$ ke dalam fungsi $f(x)$, menggantikan setiap $x$ di $f(x)$ dengan $g(x)$. Ini dibaca "f bundaran g" atau "f dari g dari x".

2. Langkah Pengerjaan
  • $f(x) = x + 2$
  • $g(x) = 3x$
  • $(f \circ g)(x) = f(g(x))$
3. Perhitungan $(f \circ g)(x) = f(g(x)) = f(3x)$
Ganti $x$ pada $f(x)$ dengan $3x$:
$f(3x) = (3x) + 2 = 3x + 2$.

Jawaban: B. Kuantitas P < Kuantitas Q. 1. Dasar Teori

Menghitung nilai komposisi fungsi pada titik tertentu. $(f \circ g)(x) \neq (g \circ f)(x)$ (tidak komutatif).

2. Langkah Pengerjaan
  • Hitung Kuantitas P: $(f \circ g)(3) = f(g(3))$.
  • Hitung Kuantitas Q: $(g \circ f)(3) = g(f(3))$.
3. Perhitungan Kuantitas P:
$g(x) = x - 1 \implies g(3) = 3 - 1 = 2$.
$f(g(3)) = f(2) = 2(2) = 4$.
Kuantitas P = 4.
Kuantitas Q:
$f(x) = 2x \implies f(3) = 2(3) = 6$.
$g(f(3)) = g(6) = 6 - 1 = 5$.
Kuantitas Q = 5.

Karena $4 < 5$, maka Kuantitas P < Kuantitas Q.

Jawaban: C. DUA pernyataan BERSAMA-SAMA cukup, SATU tidak cukup. 1. Dasar Teori

Kita perlu menentukan apakah informasi yang diberikan cukup untuk menemukan nilai $a$ dan $b$ sehingga $(f \circ f)(0)$ bisa dihitung.

2. Analisis Pernyataan Target: $(f \circ f)(0) = f(f(0))$.
$f(x) = ax + b \implies f(0) = a(0) + b = b$.
$f(f(0)) = f(b) = a(b) + b = ab + b = b(a+1)$.
Untuk menjawab, kita butuh nilai $a$ dan $b$.
  • (1) $f(1) = 3$. $a(1) + b = 3 \implies a + b = 3$. Kita punya satu persamaan dengan dua variabel. TIDAK CUKUP.
  • (2) $f(2) = 5$. $a(2) + b = 5 \implies 2a + b = 5$. Kita punya satu persamaan dengan dua variabel. TIDAK CUKUP.
  • (1) dan (2) Bersama: Kita punya sistem persamaan:
    $2a + b = 5$
    $a + b = 3$
    (Eliminasi) $a = 2$.
    (Substitusi) $2 + b = 3 \implies b = 1$.
    Kita tahu $a=2$ dan $b=1$. Kita bisa hitung $b(a+1) = 1(2+1) = 3$. CUKUP.

Jawaban: B. Kuantitas P < Kuantitas Q. 1. Dasar Teori

Menentukan $f(x)$ dari $f(x+1)$, lalu mencari $f(a)$ dan $f^{-1}(b)$.

2. Langkah Pengerjaan
  • Hitung Kuantitas P: $f(3)$.
  • Tentukan $f(x)$ dari $f(x+1)$.
  • Hitung Kuantitas Q: $f^{-1}(3)$.
3. Perhitungan Kuantitas P: $f(3)$
Kita tahu $f(x+1) = 2x - 5$.
Untuk mendapatkan $f(3)$, kita set $x+1 = 3 \implies x = 2$.
Masukkan $x=2$ ke persamaan: $f(2+1) = 2(2) - 5 \implies f(3) = 4 - 5 = -1$.
Kuantitas P = -1.

Kuantitas Q: $f^{-1}(3)$
Pertama, cari $f(x)$. Misal $u = x+1 \implies x = u-1$.
$f(u) = 2(u-1) - 5 = 2u - 2 - 5 = 2u - 7$. Jadi, $f(x) = 2x - 7$.
Sekarang cari $f^{-1}(3)$. Misal $f(a) = 3$.
$2a - 7 = 3 \implies 2a = 10 \implies a = 5$.
Jadi, $f^{-1}(3) = 5$.
Kuantitas Q = 5.

Karena $-1 < 5$, maka Kuantitas P < Kuantitas Q.

Menganalisis sifat-sifat fungsi $f(x) = \frac{x}{x-1}$.

2. Analisis Pernyataan
  • (1) $f^{-1}(x) = \frac{x}{x-1}$.
    $y = \frac{x}{x-1} \implies y(x-1) = x \implies yx - y = x \implies yx - x = y \implies x(y-1) = y \implies x = \frac{y}{y-1}$. Jadi, $f^{-1}(x) = \frac{x}{x-1}$. (BENAR)
  • (2) $(f \circ f)(x) = x$.
    $(f \circ f)(x) = f(f(x)) = f(\frac{x}{x-1}) = \frac{\frac{x}{x-1}}{\frac{x}{x-1} - 1} = \frac{\frac{x}{x-1}}{\frac{x - (x-1)}{x-1}} = \frac{\frac{x}{x-1}}{\frac{1}{x-1}} = x$. (BENAR)
  • (3) Domain $f(x)$ adalah $x \neq 1$.
    Penyebut tidak boleh nol, $x-1 \neq 0 \implies x \neq 1$. (BENAR)
  • (4) $(f \circ f \circ f)(x) = \frac{x}{x-1}$.
    $(f \circ f \circ f)(x) = f((f \circ f)(x))$. Dari (2), kita tahu $(f \circ f)(x) = x$.
    Maka $f((f \circ f)(x)) = f(x) = \frac{x}{x-1}$. (BENAR)
  • (5) $f^{-1}(2) = 1$.
    Dari (1), $f^{-1}(x) = \frac{x}{x-1}$. $f^{-1}(2) = \frac{2}{2-1} = \frac{2}{1} = 2$. (SALAH)

Dua fungsi $f(x)$ dan $g(x)$ saling invers jika $(f \circ g)(x) = x$ dan $(g \circ f)(x) = x$.

2. Analisis Pernyataan
  • (1) $f^{-1}(x) = g(x)$.
    Cari $f^{-1}(x)$. $y = 2x - 4 \implies y + 4 = 2x \implies x = \frac{y+4}{2} = \frac{1}{2}y + 2$.
    $f^{-1}(x) = \frac{1}{2}x + 2$. Ini sama dengan $g(x)$. (BENAR)
  • (2) $g^{-1}(x) = f(x)$.
    Cari $g^{-1}(x)$. $y = \frac{1}{2}x + 2 \implies y - 2 = \frac{1}{2}x \implies x = 2(y-2) = 2y - 4$.
    $g^{-1}(x) = 2x - 4$. Ini sama dengan $f(x)$. (BENAR)
  • (3) $(f \circ g)(x) = x$.
    Karena $f$ dan $g$ saling invers (dari 1 & 2), komposisi mereka pasti $x$.
    $f(g(x)) = f(\frac{1}{2}x + 2) = 2(\frac{1}{2}x + 2) - 4 = (x + 4) - 4 = x$. (BENAR)
  • (4) $(g \circ f)(5) = 5$.
    Karena $f$ dan $g$ saling invers, $(g \circ f)(x) = x$. Maka $(g \circ f)(5) = 5$. (BENAR)
3. Kesimpulan Semua 4 pernyataan benar.
